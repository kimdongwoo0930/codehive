# 💻 로컬 개발 환경 세팅

## 사전 요구사항

- Java 21
- Gradle 8.x (또는 `./gradlew` 사용)

---

## 실행 방법

```bash
cd backend
./gradlew bootRun
```

서버가 정상적으로 뜨면 `http://localhost:8080` 에서 접근 가능해요.

---

## H2 콘솔 사용법

로컬 개발 환경(`dev` 프로필)에서는 H2 인메모리 데이터베이스가 사용돼요.

### 접속

서버 실행 후 브라우저에서 아래 주소로 접속:

```
http://localhost:8080/h2-console
```

### 접속 정보

| 항목     | 값                     |
| -------- | ---------------------- |
| JDBC URL | `jdbc:h2:mem:codehive` |
| Username | `root`                 |
| Password | `1234`                 |

### 주의사항

- H2는 **인메모리** DB이기 때문에 서버를 재시작하면 데이터가 초기화돼요.
- H2 콘솔은 `dev` 프로필에서만 활성화돼요 (`prod`에서는 비활성화).

---

## 프로필 설정

| 프로필 | DB            | 용도      |
| ------ | ------------- | --------- |
| `dev`  | H2 (인메모리) | 로컬 개발 |
| `prod` | MySQL         | 배포 서버 |
